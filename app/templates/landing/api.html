api

The python file you provide must be a class named Bot that implements the methods get_update and play. That's it!

get_update(update)

Bot.get_update : (Update -> unit)

This method is used to provide your bot with the results of the previous game iteration. It will be passed an Update object. It should return None

play( )

Bot.play : (unit -> bool)

This method is called to get your bot's move. It must return True or False. True indicates that you wish to betray. False indications that you wish to remain silent.

Update

The Update object is passed as an argument to get_update( ). It contains the following attributes:

  • your_move : bool
  • opponent_move : bool
  • move : int
