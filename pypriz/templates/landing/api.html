The python file you provide must be a class named Bot that implements the method play. That's it!

play( )

Bot.play : (Update -> bool)

This method is called to get your bot's move. It must return True or False. True indicates that you wish to betray. False indications that you wish to remain silent.

Update

An Update object is passed as an argument to play( ). It contains the following attributes:

  • your_move : bool Your move last turn
  • opponent_move : bool Your opponent's move last turn
  • moves : int The number of total moves in the current match
